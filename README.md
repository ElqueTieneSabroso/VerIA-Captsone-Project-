The project aims to develop and validate a Minimum Variable Product for Veria, a mobile accessibility application designed to assist visually impaired users through AI-based interactions. The application seeks to improve independence by helping users identify objects, read labels and forms, recognize colors, and receive audio descriptions of their surroundings.

Hypothesis: If visually impaired users are provided with an accessible mobile application that uses AI to describe objects, read text, identify colors, and assist with environmental awareness through audio feedback, then their independence and confidence in daily activities will improve. This hypothesis will be evaluated through usability testing, interviews, and user feedback collected from visually impaired participants and accessibility experts. The project will measure aspects such as ease of use, clarity of audio guidance, accessibility perception, navigation simplicity, and overall usefulness of the application concept. Positive feedback and repeated user acceptance of the prototype will super the viability of developing Veria into a functional real-world application.
